## Coding Convention
    
**Class & Struct**

- 클래스/구조체 이름은 **UpperCamelCase**를 사용합니다.
(use UpperCamelCase on Classes)


**Functions & Variables**

- 함수와 변수에는 **lowerCamelCase**를 사용합니다.
  (lowerCamelCase on functions and variables)
